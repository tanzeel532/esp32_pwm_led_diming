# ESP32 LED BRIGHTNESS CONTROL (PlatformIO)

This project demonstrates basic LED control using ESP32 with the Arduino framework in PlatformIO.

## Hardware
- ESP32 development board
- LED connected to GPIO 2

## Software
- VSCode
- PlatformIO
- Arduino framework for ESP32

## Project Structure

led_blink/
 ├── src/main.cpp
 ├── platformio.ini
 └── README.md

 ## How to Build & Upload

1. Open project in VSCode
2. Click PlatformIO → Build
3. Click Upload

## Notes
- PWM frequency: 1000 Hz
- Resolution: 8-bit
- Uses ESP32 LEDC hardware PWM
